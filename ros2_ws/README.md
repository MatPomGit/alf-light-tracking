# ros2_ws

Workspace ROS 2 dla projektu `g1_light_tracking`.

## Gdzie jest pakiet

Właściwy pakiet źródłowy znajduje się w:
- `src/g1_light_tracking/`

## Budowanie

```bash
cd ros2_ws
colcon build
```

## Aktywacja środowiska

```bash
cd ros2_ws
source install/setup.bash
```

## Uruchomienie

```bash
ros2 launch g1_light_tracking prod.launch.py
```

## Dokumentacja

Dokumentacja HTML znajduje się w:
- `src/g1_light_tracking/docs/index.html`
